# Sales-and-Return-Analysis-PowerBI
Developed an interactive Power BI dashboard to track sales, profit trends, and return rates by region, category, and manager. Cleaned and transformed raw data in Excel, uncovering insights on top customers, return patterns, and profitability by shipping mode to support data-driven decisions.


## Project Objectives

Analyze overall sales and return performance

Identify high-return products and categories

Measure revenue, profit, and return impact

Support business decisions using visual insights
